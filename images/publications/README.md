## ðŸ“¸ How to Add Your Images

### Step 1: Prepare Your Images
- Take a screenshot from your paper (a key figure, diagram, or the first page)
- Crop it to roughly square dimensions
- Recommended size: 300x300 pixels or larger
- Save as JPG format

### Step 2: Name and Save
Save your images to this directory with the exact names above:

```bash
# Example:
cp ~/my-papers/reform-figure.jpg ./reform-paper.jpg
cp ~/my-papers/tutorial-diagram.jpg ./tutorial-paper.jpg
cp ~/my-papers/target-nets-fig.jpg ./target-networks-icml24.jpg
cp ~/my-papers/discount-graph.jpg ./discount-mismatch-icml23.jpg
```

### Step 3: Done!
The images will automatically appear on your home page next to each publication!

- **Keep files small**: < 500KB recommended for fast loading
- **Square or near-square** images work best (they'll be displayed at 150x150px)
- **High contrast** images work better as thumbnails
